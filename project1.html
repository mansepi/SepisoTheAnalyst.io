Olympic Games Analysis - SADC Countries

Background Information

As a data analyst i am asked by a news company to visualize data that will help readers understand how southern African development community countries (SADC) countries have performed historically in the Olympic Games for the period (1990 - 2010). This also includes details about the competitors.

Business Task

To show historical performance for different countries that have won any medals

Data collection and Transformation

The data is a public dataset available on used from 'kaggle - 120 years olympic data' , it was imported in an SQL database as a CSV and the transformations made are shown below

Queries in MySQL workbench

Data loaded in Power BI (DATA view)

Data Model

The query is loaded into Power BI, the data model created is one table.

Calculations

The following calculations were created in the Power BI reports using DAX (Data Analysis Expressions).
Number of Competitors:

Number of Competitors = DISTINCTCOUNT( 'Olympic Data for SADC 1990-2010'[ID] )
Number of Medals = COUNTROWS( 'Olympic Data for SADC 1990-2010' )

Olympic Games Analysis

The finished visualizations consist of countries that have won atleast one medal between 1990 -2010, countries with zero medals have been filtered out
